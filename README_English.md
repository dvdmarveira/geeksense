# 📋 Project Overview

**GeekSense** is an intelligent Discord bot designed to help moderators and community leaders monitor the emotional climate of university geek communities.

The bot uses Artificial Intelligence through **Google Gemini 2.5 Flash** to analyze messages in real time, identify sentiment patterns, detect toxic interactions, and provide actionable insights through automated reports.

The solution aims to promote healthier, more inclusive, and more engaging online communities for students interested in gaming, anime, technology, comics, and other geek culture topics.

## Key Features

### 🎯 Real-Time Sentiment Analysis

Automatically classifies messages as:

* Positive
* Neutral
* Negative

Each analysis includes a confidence score generated by Gemini AI.

### 🚨 Toxic Content Detection

Identifies potentially harmful content such as:

* Insults
* Harassment
* Hate speech
* Bullying
* Aggressive behavior

When toxic content is detected, moderators receive alerts through a designated administration channel.

### 📊 Weekly Community Reports

Generates automated reports containing:

* Sentiment distribution
* Most active users
* Most active channels
* Toxicity statistics
* AI-generated recommendations for community improvement

### 🤖 AI Chat Assistant

Dedicated channel for users to interact directly with Gemini AI and ask questions about:

* Anime
* Games
* Technology
* Programming
* Geek culture
* General knowledge

### 🔍 Manual Sentiment Analysis

Users can analyze any custom text using a simple command.

# 🔒 Privacy & Data Compliance

GeekSense processes community messages for sentiment and toxicity analysis. Privacy and responsible data usage were considered throughout the project design.

## Implemented Privacy Standards

### Consent & Transparency

The bot only operates on Discord servers where it has been intentionally invited and configured by administrators.

Community members are informed that message analysis is being performed.

### Data Minimization

Only the information strictly necessary for sentiment analysis and reporting is stored.

The system avoids collecting unnecessary personal information.

### User Rights

Server administrators maintain full control over bot access and configuration.

The bot can be removed at any time, immediately stopping message processing.

### Security

Sensitive credentials are protected through environment variables.

Examples include:

* Discord Bot Token
* Gemini API Key
* MongoDB Connection String

No secret keys are stored directly in the source code repository.

### Responsible AI Usage

AI-generated classifications are used as support tools for moderation and should not replace human judgment.

Moderators retain final decision-making authority regarding community actions.

# ⚙️ Getting Started (Local Development)

Follow these steps to run the project locally.

## 1. Prerequisites

Make sure you have installed:

* Git
* Node.js (v20 or higher)
* MongoDB Atlas account
* Google AI Studio account
* Discord Developer Portal account

---

## 2. Clone the Repository

```bash
git clone https://github.com/dvdmarveira/geeksense.git

cd geeksense
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Configure Environment Variables

Create a `.env` file in the project root.

```env
DISCORD_TOKEN=your_discord_bot_token

GEMINI_API_KEY=your_gemini_api_key

MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/geeksense

ADMIN_CHANNEL_ID=123456789012345678
```

## 5. Start the Application

```bash
npm start
```

If everything is configured correctly, the bot will connect to Discord and begin monitoring messages.
